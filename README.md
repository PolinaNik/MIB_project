# MIB_project
Итоговый проект (пример) курса "Машинное обучение в бизнесе"

Стек:

ML: sklearn, pandas, numpy, catboost

API: flask

Задача: предсказать откроет ли клиент срочный вклад (поле y). Бинарная классификация

Используемые признаки:

duration

contact

poutcome

Модель: logreg
